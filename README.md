# Enhanced To-Do List App

This is a simple and user-friendly to-do list application built with HTML, CSS, and JavaScript. It lets you add tasks, mark them as completed, and see them in a clean, organized layout. The design adapts to any device, so you can use it on your phone, tablet, or computer.

## Features
- **Add Tasks:** Quickly add new tasks to your list.
- **Mark as Done:** Check off tasks to mark them as completed.
- **Mobile-Friendly:** The app looks good on all devices.
- **Current Date:** Shows todayâ€™s date at the top.
- **Easy to Use:** Clean design with a simple interface.

## How to Use
1. Download or clone the repository.
2. Open the `index.html` file in your browser.
3. Add tasks, check them off when done, and enjoy!

---

## Tech Stack

- **HTML & CSS:** Structure and styling.
- **JavaScript:** Functionality for adding, completing, and deleting tasks.
- **FontAwesome:** Icons for a more user-friendly interface.
